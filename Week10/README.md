# Lab 10 – Sequential Data Generation

## Objective
Implement a generative model capable of learning patterns from sequential data and generating new sequences using RNN, LSTM, and Transformer architectures.

## Workflow

### Component I: RNN / LSTM Based Sequential Data Generation
- Load and preprocess the sequence dataset.
- Perform character-level or word-level tokenization.
- Create input-output sequence pairs.
- Design and train an RNN or LSTM based generative model.
- Generate new sequences using a seed input.

### Component II: Transformer Based Sequential Data Generation
- Use the same dataset as Component I.
- Apply word-level or subword tokenization.
- Implement positional encoding for sequence order.
- Design and train a Transformer encoder architecture.

## Expected Output
- Generated text sequences from both the RNN/LSTM and Transformer models.
